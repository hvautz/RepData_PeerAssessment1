Only column “steps” has NA values. For part 3. a new dataset with the imputed values has to be created. Therefore we distribute the mean value of the same 5-minute interval of all other days to the NA values of the steps column.

require(plyr)

## Loading required package: plyr

impute.mean <- function(x) replace(x, is.na(x), mean(x, na.rm = TRUE))
q3 <- data.table(ddply(data, ~interval, transform, steps = impute.mean(steps)))
